# Phantasma Phoenix — Website

Premium one-page website for Phantasma Phoenix. Hero section establishes
the "Obsidian & Ember" visual language for the full site.

## Stack

- **Next.js 15** (App Router, React Server Components)
- **TypeScript** (strict)
- **Tailwind CSS v4** (CSS-first config, design tokens via `@theme`)
- **Framer Motion** (entrance choreography, reduced-motion aware)
- **2D Canvas** ember particle field (DPR-capped, viewport/tab-aware rAF)

## Local development

```bash
npm install
npm run dev
